ssrc::spread::detail

ssrc::spread::detail Namespace Reference

Detailed Description

The ssrc::spread::detail namespace contains classes that are used internally by ssrc::spread classes, but are not meant for use by library users.

The classes are documented so that you may use the methods inherited by their public subclasses. Please do not directly use the classes in this namespace because they are not feature complete and their implementation may change in incompatible ways at any time.
